(1.1) CSS son las siglas de “Cascading Style Sheets”

(1.2) CSS sirve para darle mejor diseño a las paginas web y el programa que se usa es el navegador

(1.3) Es un patrón de elementos y otros términos que indican al navegador qué elementos HTML se seleccionan para aplicarles una regla que incluye los valores de las propiedades CSS.

(1.4)

(1.5) El selector CSS es el nexo de unión entre la hoja de estilos y los documentos a los que se aplique dicha hoja.

(1.6)
ID: Este tipo de selector es igual al de clase, aunque lo aplicaremos a un único elemento HTML de nuestra página. Esto se debe a que el atributo ID de HTML identifica a un elemento HTML por medio de un nombre. Por ejemplo: <h1 id="titulo">Título Principal</h1>
Clases: Los selectores de clases comienzan siempre por un . (punto). Los clientes web (navegadores) identifican este punto y lo distinguen del resto de los selectores. Por ejemplo:
<p class="primer-parrafo">Texto de un párrafo<
Universal:Nos permite aplicar estilos a todos los elementos HTML de un documento y se indica mediante el*.
* {
color:red;
}
Todos los elementos HTML tendrán un color de texto red. /p>
Etiquetas: Aplicamos propiedades CSS a una etiqueta. La misma se identifica sin utilizar < >.
Por ejemplo:
p {
color:red;
}
descendente: Con este selector seleccionamos un elemento que está dentro de otro del mismo tipo. Estos seectores siempre están formados por dos o más selectores separados entre si. El último selector será el elemento HTML al que se le aplicará las propiedades de CSS.
Por ejemplo:
p span {
color:green;
}

(1.7)

(1.8)

CSS en línea (Inline CSS): Se utiliza el atributo style directamente dentro de las etiquetas HTML para aplicar estilos específicos a ese elemento en particular.


CSS interno (Internal CSS): Se coloca el código CSS dentro de la etiqueta <style> en la sección <head>del documento HTML. Esto es útil si los estilos se aplican solo a esa página en particular.


CSS externo (External CSS): Se vincula un archivo CSS externo al documento HTML usando la etiqueta <link> en la sección <head>. Este método es preferido cuando se quiere aplicar los mismos estilos a múltiples páginas.
